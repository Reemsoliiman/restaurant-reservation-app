# Restaurant Reservation App

A Flutter app for restaurant reservations with Vendor and Customer parts, using Firebase backend.

## Structure
- `/vendor_app`: Vendor mobile app (no auth).
- `/customer_app`: Customer mobile app (with auth).

## Setup
1. Install Flutter and dependencies.
2. Set up Firebase project and add config files.
3. Run `flutter pub get` in each app folder.
4. Run the apps: `flutter run`.

## Features
- Vendor: Add restaurants, view bookings, notifications.
- Customer: Auth, browse/search, book tables.

See scenario in project description for testing.
